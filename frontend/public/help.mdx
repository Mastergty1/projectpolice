## <span id="get-started">เริ่มต้นใช้งาน</span>

<img src="/helpImage/landingPage.png" class="HelpImage" />
<small class="Caption">รูปที่ 1: หน้าจอหลักของระบบแสดงงานติดตาม</small>

เมื่อเข้าสู่แอพพลิเคชั่น หน้าหลักจะทำการแสดงผลข้อมูลงานติดตามต่างๆ ซึ่งผุ้ใช้งานสามารถทำการเลือกขอบเขตการแสดงผลเพื่อค้นหางานติดตามที่ต้องการได้ ซึ่งสามารถเรียกใช้งานได้จากตัวเลือกด้านบนรายชื่องานติดตาม

<img src="/helpImage/AddTaskButton.png" class="HelpImage" />
<small class="Caption">รูปที่ 2: ปุ่มเพิ่มงานติดตาม</small>

หากต้องการเพิ่มงานติดตาม ให้ไปที่ปุ่ม <span class="Button">+ เพิ่มงานติดตาม</span>ที่ด้านบนขวา และหากต้องการกลับมาที่หน้าจอหลัก ให้คลิกที่ไอคอนที่ด้านบนซ้าย

## <span id="all-tasks">งานติดตาม</span>

<img src="/helpImage/TaskList.png" class="HelpImage" />
<small class="Caption">รูปที่ 3: ส่วนแสดงงานติดตามต่างๆ</small>

งานติดตามที่แสดงผลที่หน้าจอหลักนั้นแบ่งออกเป็นทั้งหมด 2 ประเภท ได้แก่ **งานติดตามเร่งด่วน** และ **งานติดตามทั้งหมด** ซึ่งมีรายละเอียดแยกย่อยดังนี้

>- **งานติดตามเร่งด่วน** 
>
> > งานติดตามเร่งด่วนจะแสดงผลงานที่มีกำหนดรายงานผลภายใน 7 วันขึ้นมาก่อนเป็นอันดับแรก รวมไปถึงงานอื่นๆ ที่ถูกระบุเอาไว้เป็นงานติดตามเร่งด่วนเมื่อทำการสร้าง
>

>- **งานติดตามทั้งหมด**

งานติดตามท้งหมดจะแสดงผลงานติดตามทั้งหมดโดยเรียงจากงานที่ใกล้กำหนดรายงานผลก่อน 

<img src="/helpImage/TaskDisplay.png" class="HelpImage" />
<small class="Caption">รูปที่ 4: รายละเอียดงานติดตามที่แสดงผลในหน้านี้</small>


โดยงานติดตามเหล่านี้นั้นจะถูกแยกประเภทสีตามระยะเวลาที่คงเหลืออยู่
>>
>> - <span class="Grey">สีเทา</span> คืองานติดตามที่เลยกำหนดรายงานไปแล้ว
>> - <span class="Red">สีแดง</span> คืองานติดตามที่มีกำหนดรายงานภายในวันนี้
>> - <span class="Orange">สีส้ม</span> คืองานติดตามที่มีกำหนดรายงานภายในวันพรุ่งนี้
>> - <span class="Yellow">สีเหลือง</span> คืองานติดตามที่มีกำหนดรายงานภายในสัปดาห์นี้
>> - <span class="Green">สีเขียว</span> คืองานติดตามที่มีกำหนดรายงานนอกเหนือไปจากนัั้น

ภายในงานติดตามแต่ละงานนั้น ด้านซ้ายจะแสดงวันที่กำหนดรายงาน ต่อด้วยรายละเอียดและผู้รับผิดชอบ ส่วนพื้นที่ด้านขวาจะแสดงผลสถานะของงานติดตามนั้นๆ 

>สถานะของงานติดตามนั้นประกอบไปด้วย
>>- <span class="Yellow"> กำลังติดตาม </span>
>>- <span class="Red"> เกิดปัญหา </span>
>>- <span class="Green"> เสร็จสิ้น </span> เมื่อตั้งค่างานติดตามให้เป็นสถานะนี้ งานติดตามจะถูกลบออกไปจากการติดตาม

นอกจากนี้ สามารถคลิกปุ่ม <span class="Button">รายละเอียด</span>เพิ่อดูรายละเอียดเพิ่มเติมเกี่ยวกับงานติดตามนั้นๆ ได้

## <span id="task-details">รายละเอียด</span>

<img src="/helpImage/TaskDisplay.png" class="HelpImage" />
<small class="Caption">รูปที่ 5: หน้าแสดงรายละเอียดงานติดตามอย่างละเอียด</small>

ที่หน้านี้จะทำการแสดงผลรายละเอียดเกี่ยวกับงานติดตาม ประกอบไปด้วย

>- **รายละเอียดข้อมูลพื้นฐาน**
>> เฉกเช่นเดียวกับที่แสดงผลในหน้าหลัก ผู้ใช้สามารถเปลี่ยนสถานะของงานติดตามหรือเขียนบันทึกเพิ่มเติมเกี่ยวกับงานติดตามชิ้นนั้นๆ เพื่อตรวจสอบในภายหลังได้

>- **รายละเอียดจากเอกสาร (ข้อความเต็ม)**
>> เป็นเนื้อหาที่ถอดความมาได้จากงานเอกสาร ซึ่งจะแสดงผลงานมอบหมายที่เฉพาะเจาะจงกับผู้ที่ได้รับมอบหมายแต่ละคน และสามารถคลิกปุ่ม <span class="Button">เปิดดูไฟล์เอกสารต้นฉบับ</span>เพื่อดูเอกสารที่ใช้ถอดความมาได้

ที่ด้านล่างสุดของหน้า สามารถเลือก <span class="Yellow">แก้ไขข้อมูล</span>หรือ <span class="Red">ลบงานติดตามนี้</span>ได้
